initialize Node application =>   npm init -y

express framwork => npm install express

install nodemon => npm install nodemon  : to run the application 

install dotenv => npm install dotenv : to connect to connect to .env file automatically

install mongoose => npm install mongoose : Mongoose is an object document modeling (ODM) layer that sits on top of Node's MongoDB driver. 

install bcryptjs => npm install bcryptjs:  to encrypt our password in the databases

install JWT => npm install jsonwebtoken : a mechanism to verify the owner of some JSON data.

generate secrete code => openssl rand -base64 32

install cookie-parser => npm install cookie-parser : to setting token into cookie
